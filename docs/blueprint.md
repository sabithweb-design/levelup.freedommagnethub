# **App Name**: LearnVista Landing

## Core Features:

- Dynamic Program Page: Renders specific program details, including title, subtitle, demo video, image gallery, and testimonials, fetched from the 'programs' Firestore collection using the '/program/:id' route.
- Program Overview & Demo: Displays the program's main title, subtitle, and integrates an embedded YouTube demo video with the Plyr player UI.
- Multimedia Content Display: Showcases a dedicated image gallery for the program and seamlessly embeds multiple YouTube videos for testimonials.
- Image Testimonial Section: Effectively displays five static image testimonials to provide social proof and credibility for the program.
- Time-Sensitive Call-to-Action: Features a prominent 'Join Now' button with a dynamic countdown timer to create urgency and encourage user engagement.
- Responsive Design System: Ensures the landing page layout and content adapt flawlessly for optimal viewing on desktop, tablet, and mobile devices.

## Style Guidelines:

- Primary color: A deep, professional blue (#2F6DBE), symbolizing trust and knowledge, providing a strong anchor for headings and interactive elements.
- Background color: A subtle, desaturated light blue (#E7EEF7) that offers a clean, expansive canvas, ensuring content readability and a modern aesthetic.
- Accent color: A vibrant aqua (#2EC6D1), used strategically for calls to action, highlights, and key interactive components to draw user attention.
- Headlines will use 'Space Grotesk' (sans-serif) for a modern, slightly technical yet clean appeal, enhancing readability of key program titles.
- Body text will utilize 'Inter' (sans-serif), chosen for its excellent legibility and neutrality across various screen sizes, supporting extensive program details and testimonials.
- Clean, line-based vector icons will be employed to complement the modern UI, ensuring visual clarity and enhancing user experience without adding clutter.
- A responsive, grid-based layout will define content sections, maximizing readability and ensuring seamless adaptation to mobile and desktop viewports, with ample whitespace.
- Subtle, performance-optimized animations will be used for state changes or element transitions, contributing to a smooth user experience without hindering fast loading times.
